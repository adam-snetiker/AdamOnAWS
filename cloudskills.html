Project 4: Setting up Projects in GitHub

Under the Projects tab in GitHub, I created a new project using the Basic Kanban template to track the status of tasks. This board includes three columns:

  • To do
  • In progress
  • Done

Other capabilities include:

  • Add a new column
  • Drag and drop cards between columns
  • Search for and add issues to projects or repositories
  • Automate various board tasks
  • Archive cards or full columns
