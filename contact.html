The back-end of this form is built on a "serverless" architecture. Below are the services and micro-tasks I used to make it work.


AWS Lambda

  • Created a function that handles the sending and receiving of email using SES.

Amazon Simple Email Service (SES)

  • Verified the email address to be used for inquiries. This step was already completed because I created the email address adam@adamonaws.com at the beginning of the project.

AWS Identity and Access Management (IAM)

  • Set up a role that allows SES and Lambda to communicate.
  • Attached the policies AmazonSESFullAccess and AWSLambdaExecute.

Amazon API Gateway

  • Built a REST API that integrates with the Lambda function created earlier.
  • Enabled Cross-Origin Resource Sharing (CORS) so requests can be made from any version of my website, regardless of what protocol or sub-domain is accessing this page. For example, www.adamonaws.com is a different origin than adamonaws.com, and https://adamonaws.com is a different origin than http://adamonaws.com.

JavaScript

Added a script to the contact form page that will do two things:
  1. Send the form data to the API Gateway when the user presses the submit button
  2. Execute the Lamba function that will tell SES to send the form inputs to my email address.
